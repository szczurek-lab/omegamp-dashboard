# OmegAMP Interactive Dashboard

Interactive companion to the OmegAMP manuscript. Explore antimicrobial activity, safety, membrane mechanism, and target binding data for 217 peptides generated by the OmegAMP diffusion framework.

**[Open Dashboard](https://szczurek-lab.github.io/omegamp-dashboard/)** (password provided by the corresponding author)

## What's in the dashboard

The dashboard presents wet-lab validation results for 95 de novo peptides, 59 analogs from inactive prototypes, 40 LPS-binding analogs, 10 DNA-binding analogs, 11 prototypes, and 2 antibiotic controls (Polymyxin B, Levofloxacin), tested against 20 bacterial strains including 8 multidrug-resistant clinical isolates.

### Tabs

| Tab | What it shows |
|-----|---------------|
| **Activity** | MIC heatmap across all 20 strains with CC50, HC50, and Safety Window columns. Sortable by family, Gram-negative/positive potency, or any individual strain. |
| **Physicochemical Landscape** | Charge, hydrophobicity, and other descriptors plotted against each other, colored by activity or safety metrics. |
| **Membrane Activity** | NPN (outer membrane permeabilization) and DiSC3(5) (cytoplasmic membrane depolarization) AUC-vs-MaxRel plots, plus an NPN-vs-DiSC MaxRel cross-plot for mechanistic profiling. |
| **Target Binding** | LPS neutralization (BC probe displacement at 2, 8, 32 uM +/- Ca2+) and DNA binding (deltaA from CD spectrometry). |
| **Structure** | Circular dichroism secondary structure (BeStSel deconvolution) across four solvent conditions (H2O, MeOH, SDS, TFE). |
| **Safety** | MIC vs CC50/HC50 scatter plots for Safety Window assessment. |
| **Compare** | Side-by-side comparison table of selected peptides. |

## How to use it

**Browse** -- Switch between tabs to explore different assay results. Each peptide appears as a dot (scatter plots) or row (heatmaps). Hover for details.

**Select** -- Click any peptide to select it. Click again to deselect. Select up to 10 peptides to compare them in the Compare tab. Selected peptides are highlighted with green circles across all tabs, and shown as pills in the yellow bar below the header.

**Filter** -- Use the filter bar to show/hide experiment groups (de novo, inactive-to-active, LPS-binding, DNA-binding) or peptide categories (analog, prototype, control). Use the Families dropdown to isolate specific prototype families.

**Adjust MIC summary** -- Toggle between geometric mean, MIC50 (median, default), and MIC90 across all views. The "Active if MIC <=" dropdown controls what counts as an active strain (default: 4 uM).

**Sort the heatmap** -- Click preset buttons (Family, Gram- MIC, Gram+ MIC, Strains) or click any column header to sort. Click the "Gram (-)" or "Gram (+)" header row to sort by Gram-negative or Gram-positive potency.

**Zoom scatter plots** -- Click and drag to zoom into a region. Click "Reset zoom" to return.

**Expand membrane plots** -- Click the expand button next to any membrane plot title to view it at full width.

**Help panels** -- Click the "About" buttons next to assay section headers to open a slide-out panel explaining the assay, its metrics, and how to interpret the results.

**Export** -- Use "Export CSV" in the header to download all filtered data, or "Export selected" in the Compare tab for just the selected peptides.

## Assay overview

| Assay | What it measures | Key metric |
|-------|-----------------|------------|
| **MIC** | Minimum inhibitory concentration against 20 strains (14 Gram-, 5 Gram+, 1 lab ref) | Lower = more potent |
| **CC50** | Cytotoxicity against HEK293T cells (non-linear regression of dose-response) | Higher = safer |
| **HC50** | Hemolysis of human erythrocytes (non-linear regression of dose-response) | Higher = safer |
| **NPN** | Outer membrane permeabilization (*A. baumannii* ATCC 19606) | MaxRel (%), AUC |
| **DiSC3(5)** | Cytoplasmic membrane depolarization (*A. baumannii* ATCC 19606) | MaxRel (%), AUC |
| **LPS binding** | BC probe displacement (competitive LPS binding) | % displacement |
| **DNA binding** | CD spectrometry (peptide-DNA complex formation) | deltaA |
| **CD / BeStSel** | Secondary structure in 4 solvents | % alpha-helix, beta, turn, other |

CC50 and HC50 values above 128 uM are right-censored as >128, following standard AMP literature practice for values requiring extrapolation beyond the tested concentration range.

## Reproducing the manuscript figures

This repository contains the figure notebooks (`notebooks/`) and the
scripts that regenerated the datasets they visualise
(`scripts/reproduce/`).  The generative model itself lives in a
separate repository (`generative-modelling-amp`).

### Layout

| Path | Contents |
|------|----------|
| `notebooks/figure2_conditioning.ipynb`, `notebooks/figureS1_conditioning.ipynb` | Property conditioning figures |
| `notebooks/figure3_analog.ipynb`, `notebooks/figureS2_analog.ipynb` | Analog generation figures |
| `notebooks/figure4_denovo.ipynb`, `notebooks/figure5_analog.ipynb`, `notebooks/figure6_mechanism.ipynb`, `notebooks/figureS3_analog-wet.ipynb`, `notebooks/figureS4_mechanism.ipynb` | Wet-lab figures |
| `scripts/reproduce/` | Shell scripts that regenerate the model outputs read by Figure 2 and Figure 3 notebooks |
| `scripts/original/` | Original, unedited generation scripts used during development (kept for provenance) |
| `amp_similarity/` | AMP-specific pairwise similarity scoring (used by Figure 3 Panel B) |

### Getting the data and rendered figures

Several components are not tracked in this public repository until
manuscript publication:

- **Rendered figures** -- PNG/PDF/SVG outputs of each figure notebook.
  Notebooks save to a local `figures/` directory when run.
- **Model-generation outputs for Figures 2 and 3** -- conditioning
  grids, analog parameter sweeps, benchmark FASTAs, APEX predictions.
  Expected at `data/figure2/` and `data/figure3/`.
- **Wet-lab assay CSVs for Figures 4--6** -- MIC, CC50, HC50, NPN,
  DiSC3(5), BeStSel, LPS binding, DNA binding, reference table.
  Expected at `data/`.

These will be made publicly available upon publication of the
manuscript.  In the meantime, access can be requested from the
corresponding authors.

Alternatively, the Figure 2 and 3 data can be regenerated from scratch
with a GPU, the `generative-modelling-amp` repo, and APEX -- see
"Regenerate datasets" below.

### Prerequisites for regeneration

| Dependency | What it is | Where to get it |
|---|---|---|
| `generative-modelling-amp` | OmegAMP model code + checkpoint | Internal repo |
| APEX | MIC prediction tool | Internal (`/raid/battleamp_root/…`) |

### 1. Set environment variables

```bash
export OMEGAMP_DIR=/path/to/generative-modelling-amp
export APEX_DIR=/path/to/apex            # only needed for APEX prediction steps
```

The checkpoint is expected at `${OMEGAMP_DIR}/data/generative_model.ckpt`.
Override with `export CHECKPOINT=/other/path/model.ckpt` if needed.

### 2. Run figure notebooks

Open any notebook in `notebooks/` with JupyterLab.  The notebooks read
pre-generated data from `data/` (see "Getting the data and rendered
figures" above).

### 3. Regenerate datasets (optional)

Each script in `scripts/reproduce/` regenerates one dataset from scratch.

```bash
# Figure 2 -- property conditioning
bash scripts/reproduce/gen_figure2_conditioning.sh   # ~2-4 h on one GPU
bash scripts/reproduce/gen_figure2_apex.sh           # APEX predictions

# Figure 3 -- analog parameter sweep & flavor comparison
bash scripts/reproduce/gen_figure3_sweep.sh
bash scripts/reproduce/gen_figure3_flavors.sh

# Wet-lab candidate generation
bash scripts/reproduce/gen_wetlab.sh
bash scripts/reproduce/gen_wetlab_motifs.sh          # template/motif runs
```

## Building the dashboard locally

Requires Python 3.8+ with [modlAMP](https://modlamp.org/).

```bash
pip install modlamp
python build_dashboard.py --data-dir data --help-dir help -o docs/index.html
```
